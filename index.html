String Methods
Array Methods
Object Methods

trim()

Description: Removes whitespace from both ends of a string.

Example: " Hello ".trim()

Result: "Hello"

toLowerCase()

Description: Converts all characters to lowercase.

Example: "Hello World".toLowerCase()

Result: "hello world"

toUpperCase()

Description: Converts all characters to uppercase.

Example: "Hello World".toUpperCase()

Result: "HELLO WORLD"

charAt()

Description: Returns the character at a specified index.

Example: "Hello".charAt(1)

Result: "e"

charCodeAt()

Description: Returns the Unicode value of the character at a specified index.

Example: "Hello".charCodeAt(1)

Result: 101

concat()

Description: Combines two or more strings.

Example: "Hello".concat(" ", "World")

Result: "Hello World"

includes()

Description: Checks if a string contains a specified substring.

Example: "Hello World".includes("World")

Result: true

indexOf()

Description: Returns the index of the first occurrence of a specified substring.

Example: "Hello World".indexOf("World")

Result: 6

lastIndexOf()

Description: Returns the index of the last occurrence of a specified substring.

Example: "Hello Hello".lastIndexOf("Hello")

Result: 6

match()

Description: Retrieves the matches of a string against a regular expression.

Example: "Hello 123".match(/\d+/)

Result: ["123"]

replace()

Description: Replaces occurrences of a substring with a new substring.

Example: "Hello World".replace("World", "Universe")

Result: "Hello Universe"

search()

Description: Searches for a match between a string and a regular expression.

Example: "Hello World".search(/World/)

Result: 6

slice()

Description: Extracts a portion of a string between two indices.

Example: "Hello World".slice(0, 5)

Result: "Hello"

split()

Description: Splits a string into an array of substrings based on a delimiter.

Example: "Hello World".split(" ")

Result: ["Hello", "World"]

substr()

Description: Returns a specified number of characters from a string, starting at a specified index.

Example: "Hello World".substr(6, 5)

Result: "World"

substring()

Description: Extracts characters from a string between two indices.

Example: "Hello World".substring(6, 11)

Result: "World"

repeat()

Description: Returns a new string with a specified number of copies of the original string.

Example: "Hello".repeat(3)

Result: "HelloHelloHello"

startsWith()

Description: Checks if a string starts with a specified substring.

Example: "Hello World".startsWith("Hello")

Result: true

endsWith()

Description: Checks if a string ends with a specified substring.

Example: "Hello World".endsWith("World")

Result: true

trimStart()

Description: Removes whitespace from the beginning of a string.

Example: " Hello ".trimStart()

Result: "Hello "

trimEnd()

Description: Removes whitespace from the end of a string.

Example: " Hello ".trimEnd()

Result: " Hello"

push()

Description: Adds one or more elements to the end of an array.

Example: [1, 2].push(3)

Result: [1, 2, 3]

pop()

Description: Removes the last element from an array.

Example: [1, 2, 3].pop()

Result: 3

shift()

Description: Removes the first element from an array.

Example: [1, 2, 3].shift()

Result: 1

unshift()

Description: Adds one or more elements to the beginning of an array.

Example: [1, 2].unshift(0)

Result: [0, 1, 2]

concat()

Description: Combines two or more arrays into a new array.

Example: [1, 2].concat([3, 4])

Result: [1, 2, 3, 4]

join()

Description: Joins all elements of an array into a string.

Example: [1, 2, 3].join('-')

Result: "1-2-3"

slice()

Description: Extracts a section of an array and returns it as a new array.

Example: [1, 2, 3, 4].slice(1, 3)

Result: [2, 3]

splice()

Description: Adds or removes elements from an array at a specified index.

Example: [1, 2, 3].splice(1, 1, 4, 5)

Result: [1, 4, 5, 3]

sort()

Description: Sorts the elements of an array in place.

Example: [3, 1, 2].sort()

Result: [1, 2, 3]

reverse()

Description: Reverses the order of elements in an array.

Example: [1, 2, 3].reverse()

Result: [3, 2, 1]

forEach()

Description: Executes a provided function once for each array element.

Example: [1, 2, 3].forEach(el => console.log(el))

Result: 1, 2, 3 (printed to the console)

map()

Description: Creates a new array with the results of calling a function on every element.

Example: [1, 2, 3].map(el => el * 2)

Result: [2, 4, 6]

filter()

Description: Creates a new array with all elements that pass a test.

Example: [1, 2, 3].filter(el => el > 1)

Result: [2, 3]

reduce()

Description: Applies a function against an accumulator and each element in the array to reduce it to a single value.

Example: [1, 2, 3].reduce((acc, val) => acc + val, 0)

Result: 6

find()

Description: Returns the first element that satisfies a provided testing function.

Example: [1, 2, 3].find(el => el > 1)

Result: 2

findIndex()

Description: Returns the index of the first element that satisfies a provided testing function.

Example: [1, 2, 3].findIndex(el => el > 1)

Result: 1

some()

Description: Tests whether at least one element in the array passes the test implemented by the provided function.

Example: [1, 2, 3].some(el => el > 2)

Result: true

every()

Description: Tests whether all elements in the array pass the test implemented by the provided function.

Example: [1, 2, 3].every(el => el > 0)

Result: true

includes()

Description: Determines whether an array includes a certain value among its entries.

Example: [1, 2, 3].includes(2)

Result: true

toString()

Description: Returns a string representing the array.

Example: [1, 2, 3].toString()

Result: "1,2,3"

copyWithin()

Description: Shallow copies part of an array to another location in the same array and returns it without modifying its length.

Example: [1, 2, 3, 4].copyWithin(0, 2)

Result: [3, 4, 3, 4]

fill()

Description: Changes all elements in an array to a static value from a start index (default is 0) to an end index (default is array.length).

Example: [1, 2, 3].fill(0, 1, 2)

Result: [1, 0, 3]

flat()

Description: Creates a new array with all sub-array elements concatenated into it recursively up to the specified depth.

Example: [1, [2, [3, [4]]]].flat(2)

Result: [1, 2, 3, [4]]

flatMap()

Description: First maps each element using a mapping function, then flattens the result into a new array.

Example: [1, 2, 3].flatMap(x => [x, x * 2])

Result: [1, 2, 2, 4, 3, 6]

from()

Description: Creates a new array instance from an array-like or iterable object.

Example: Array.from('foo')

Result: ["f", "o", "o"]

isArray()

Description: Determines whether the passed value is an array.

Example: Array.isArray([1, 2])

Result: true

keys()

Description: Returns an array of a given object's own enumerable property names.

Example: Object.keys({a: 1, b: 2})

Result: ["a", "b"]

values()

Description: Returns an array of a given object's own enumerable property values.

Example: Object.values({a: 1, b: 2})

Result: [1, 2]

entries()

Description: Returns an array of a given object's own enumerable property [key, value] pairs.

Example: Object.entries({a: 1, b: 2})

Result: [["a", 1], ["b", 2]]

assign()

Description: Copies the values of all enumerable own properties from one or more source objects to a target object.

Example: Object.assign({a: 1}, {b: 2})

Result: {a: 1, b: 2}

freeze()

Description: Freezes an object, preventing new properties from being added, and marking all existing properties as read-only.

Example: Object.freeze({a: 1})

Result: {a: 1} (but properties cannot be modified or deleted)

seal()

Description: Seals an object, preventing new properties from being added and marking all existing properties as non-configurable.

Example: Object.seal({a: 1})

Result: {a: 1} (but existing properties can be modified)

getPrototypeOf()

Description: Returns the prototype (i.e., the internal [[Prototype]] property) of the specified object.

Example: Object.getPrototypeOf({})

Result: {} (the prototype object)

setPrototypeOf()

Description: Sets the prototype (i.e., the internal [[Prototype]] property) of a specified object.

Example: Object.setPrototypeOf({}, Object.prototype)

Result: The specified object with a new prototype.

create()

Description: Creates a new object with the specified prototype object and properties.

Example: Object.create({a: 1})

Result: {} (with prototype {a: 1})

is()

Description: Determines whether two values are the same value.

Example: Object.is('foo', 'foo')

Result: true

hasOwnProperty()

Description: Returns a boolean indicating whether the object has the specified property as its own property.

Example: ({a: 1}).hasOwnProperty('a')

Result: true

defineProperty()

Description: Adds a property to an object, or modifies an existing property on an object, and returns the object.

Example: Object.defineProperty({}, 'a', {value: 1})

Result: {a: 1}

defineProperties()

Description: Adds multiple properties to an object, or modifies existing properties on an object, and returns the object.

Example: Object.defineProperties({}, {a: {value: 1}, b: {value: 2}})

Result: {a: 1, b: 2}

getOwnPropertyDescriptor()

Description: Returns a property descriptor for a named property of an object.

Example: Object.getOwnPropertyDescriptor({a: 1}, 'a')

Result: {value: 1, writable: true, enumerable: true, configurable: true}

getOwnPropertyDescriptors()

Description: Returns an object describing the properties of a given object.

Example: Object.getOwnPropertyDescriptors({a: 1})

Result: {a: {value: 1, writable: true, enumerable: true, configurable: true}}

prototype

Description: The prototype property allows you to add new properties and methods to a constructor function.

Example: function Person() {} Person.prototype.sayHello = function() { return 'Hello'; }

Result: Person objects will now have a sayHello method.
